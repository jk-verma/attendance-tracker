## Getting Started

1. Open `index.html` in any modern browser.
2. Select the employee type, pick a date, and enter punch times.
3. Click **Save Punch** to record and evaluate the entry.
4. Use **Filter Month** to view records for a specific month and see the summary panel.
5. Export your data via QR code or CSV from the action buttons at the bottom.
