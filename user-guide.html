Overview

The Monthly Attendance Punch Tracker helps you record daily punch-in and punch-out times for Teaching and Non-Teaching employees. It automatically evaluates each entry against the applicable attendance rules and marks the record as Compliant or Non-Compliant.

All data is stored locally in your browser — nothing is sent to a server.

Recording a Punch

  1. Select the Employee Type (Teaching or Non-Teaching) using radio buttons.
  2. Pick the Date using the date picker.
  3. Set the Punch-In and Punch-Out times.
  4. Choose the Required Daily Hours (default 8 h 30 m).
  5. If the day is a closed holiday or special leave, set the corresponding dropdown to Yes.
  6. If applicable, set Official Tour to Local Station or Out Station.
  7. Click Save Punch. The record appears in the table below with its compliance status and reason.

Filtering & Monthly Summary

Use the Filter Month picker to view records for a specific month. When a month is selected, a summary panel appears showing counts of compliant, non-compliant, grace, relaxation, late-compensation, and official-tour entries.

Editing & Deleting Records

  • Edit — Click the edit icon in the table row to reload the record into the form, make changes, and save again.
  • Delete a single record — Click the delete button on the table row.
  • Delete in bulk — Use the Delete Records button at the bottom to remove all records for the filtered month or the entire dataset.

Import & Export

  • Export as QR Code — Generate a QR code containing records for the filtered month or all records, and close the QR display using Close QR.
  • Export as CSV — Download a CSV file for the filtered month or all records.
  • Import via QR Scan — Use your camera to scan a previously exported QR code.
  • Import via QR Image — Upload a saved QR code image to import records.
  • Import via CSV — Upload a CSV file to import records.

Tips

  • Records are stored in your browser's local storage. Clearing browser data will erase all records — use export to back up.
  • Teaching and Non-Teaching follow different attendance rule sets; make sure the correct employee type is selected.
  • Semimonthly relaxation is limited to 2 entries per half-month for both Teaching and Non-Teaching.
